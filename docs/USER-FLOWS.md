# سيناريوهات الاستخدام والمسارات — تطبيق مركز العلاج الفيزيائي

## 1. حجز موعد (مع أو بدون نقل)

1. المريض أو الموظف يفتح شاشة حجز موعد.
2. يختار الفرع (إن وجد)، ثم الخدمة (جلسة أولى / متابعة / نوع العلاج).
3. يختار التاريخ ويظهر له الأوقات المتاحة حسب جدول الطبيب.
4. يختار الوقت والطبيب (أو يُعيّن تلقائياً).
5. إن كان المريض يريد نقل: يفعّل "أحتاج نقل" ويُدخل عنوان الالتقاط والوقت المطلوب.
6. يؤكد الحجز → يُنشأ **موعد** وحالة "مؤكد" أو "معلق".
7. إن وُجد نقل → يُنشأ **طلب نقل** مرتبط بالموعد.
8. يُرسل إشعار للمريض (SMS/Email/Push) بتأكيد الموعد (ورقم طلب النقل إن وجد).

## 2. استلام طلب النقل (السائق)

1. السائق يفتح تطبيق السائق ويرى قائمة الطلبات المعينة له (أو المتاحة).
2. يختار طلباً → يظهر تفاصيل: المريض، عنوان الالتقاط، وقت الموعد، عنوان المركز.
3. يضغط "بدء الرحلة" → تتحدث الحالة إلى "في الطريق".
4. (اختياري) يشارك موقعه للمركز أو للمريض.
5. عند الوصول للمريض يضغط "وصلت للمريض".
6. عند الوصول للمركز يضغط "إنهاء الرحلة" → الحالة "منجز".

## 3. رفع أشعة قبل الموعد

1. المريض يدخل إلى "ملفي" أو "مواعيدي" ويختار الموعد القادم.
2. يضغط "رفع صور أو تقارير".
3. يختار نوع المرفق (أشعة، تقرير، صورة) ويُرفع الملف.
4. النظام يربط الملف تلقائياً بالمريض والموعد القادم.
5. الطبيب قبل أو أثناء الجلسة يفتح ملف المريض ويرى المرفقات المعتمدة لهذه الجلسة.

## 4. تسجيل جلسة وإكمال الخطة العلاجية

1. الموعد يحصل → الموظف يغيّر الحالة إلى "منجز" أو يفتح "تسجيل الجلسة".
2. يفتح نموذج الجلسة: الإجراءات، الملاحظات، المرفقات (إن رُفعت أثناء الجلسة).
3. يُحدّث **الخطة العلاجية**: عدد الجلسات المكتملة +1.
4. إن انتهت الخطة أو تحتاج تعديل → يحدّث الطبيب الخطة (جلسات إضافية أو أهداف جديدة).

## 5. التواصل مع المريض

1. من لوحة الموظف: يختار المريض ثم "إرسال رسالة".
2. يكتب النص ويُرسل → تُحفظ في **messages** ويُرسل إشعار للمريض (Push/Email حسب الإعداد).
3. المريض يفتح التطبيق ويرى الرسائل في "رسائلي" ويُمكنه الرد.
4. كل الرسائل مسجلة مع التاريخ والمرسل (للمراجعة والشكاوى).

## 6. التمارين المنزلية الذكية (ما بعد الجلسة)

1. بعد انتهاء الجلسة، الطبيب يفتح "تمارين منزلية" من واجهة تسجيل الجلسة أو من ملف المريض.
2. يختار فيديوهات من **المكتبة** (exercise_library) — حسب نوع العلاج والهدف (مرونة، قوة، إلخ).
3. يحدد للمريض **مهام يومية** (أو أسبوعية): كل فيديو كم مرة، وتاريخ الاستحقاق أو تكرار.
4. النظام يُنشئ **patient_exercise_assignments** ويُرسل للمريض إشعار (Push/In-app) بقائمة التمارين.
5. المريض يفتح التطبيق في قسم "تماريني" ويرى قائمة المهام مع روابط الفيديوهات.
6. بعد تنفيذ التمرين، المريض يضغط **"تأكيد الإنجاز"** → يُسجَّل **exercise_completions** مع completed_at و confirmed_by_patient = true.
7. الطبيب/النظام يطّلع على نسبة الإنجاز (كم مهام أُنجزت من المطلوب) في الجلسة القادمة أو من لوحة المتابعة.
