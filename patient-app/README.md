# تطبيق المريض — مركز العلاج الفيزيائي

تطبيق React Native + Expo للمرضى. يتصل بالـ API الحالي عبر `EXPO_PUBLIC_API_URL`.

## التشغيل

```bash
cd patient-app
npm install
# إعداد .env: EXPO_PUBLIC_API_URL=http://localhost:3000
npx expo start
```

- **iOS:** `npm run ios` أو مسح QR من تطبيق Expo.
- **Android:** `npm run android` أو مسح QR.

## تسجيل الدخول

- **رقم الجوال:** رقم المستخدم (المريض) المسجّل في النظام.
- **كلمة المرور:** كلمة مرور الحساب.

يجب أن يكون دور المستخدم `patient` وإلا تظهر رسالة "هذا التطبيق للمرضى فقط".

## الواجهات

- **الرئيسية:** ترحيب، الموعد القادم، إحصائيات مصغرة، تمارين اليوم، آخر إشعار.
- **مواعيدي:** فلتر (القادمة / المكتملة / الكل)، قائمة المواعيد، تفاصيل الموعد مع النقل إن وجد.
- **تماريني:** شريط تقدم، أيام الأسبوع، بطاقات التمارين مع تأشير الإكمال (POST /patient-exercises/:id/completions).
- **تقدمي:** درجة التعافي، رسم بياني من GET /patients/:id/progress، ملخص آخر جلسة.

## الإشعارات

عند تسجيل الدخول يتم طلب صلاحية الإشعارات وإرسال رمز Expo Push إلى الـ API عبر `PATCH /patients/:id/push-token`.

## الأصول

إذا لم تكن مجلد `assets` مملوءاً، انسخ من قالب Expo أو أضف `icon.png` و `splash-icon.png` يدوياً.
